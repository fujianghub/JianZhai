# 简斋 · 简单版

> **简斋·开发指南** 是 JianZhai 项目的官方架构与上手说明（当前 **v0.9**）。  
> 本文面向第一次接触项目的人；想深入实现细节请阅读 → [详细版](./detailed/)。

---

## 简斋是什么

**简斋（JianZhai）** 是一套本地部署的「个人知识库 + 公开博客」一体化 Web 应用：

- **写一次，两处用**：同一份内容既有私人笔记（`raw_content`），也可发布为博客（`published_content`）
- **数据在本地**：正文与附件存在你自己的 PostgreSQL 与磁盘上
- **功能合一**：写作、版本、搜索、导出、公开阅读、AI 助手都在同一个系统里完成

Monorepo 结构：`backend/`（Django）+ `frontend/`（React），默认端口 **8002 / 3001**。

## 核心概念

| 概念 | 说明 |
|------|------|
| **知识库（KB）** | 文档的顶层容器，例如「读书笔记」「项目 A」 |
| **文件夹（Folder）** | KB 内可多层嵌套的目录 |
| **文档（Document）** | 单篇内容；含私人版与发布版两套正文 |
| **发布 / 可见性** | `status=published` 且 `visibility=public` 时出现在博客 |
| **双向链接** | 文中 `@文档名` 可链到另一篇；反链区显示「谁引用了我」 |
| **乐观并发** | 保存时带 `version`，冲突会提示并拉取最新，避免互相覆盖 |

## 功能一览

| 模块 | 你能做什么 |
|------|------------|
| 知识库 | 建库、树形目录、拖拽排序、软删除 |
| 编辑器 | **富文本 / Markdown / HTML** 三种模式；数学、分栏、Mermaid、文档卡片等 |
| HTML 阅读 | 发布版为完整 HTML 时，博客用原位 iframe 保留原样式 |
| 版本 | 快照、diff、回滚（每文档保留 100 个版本） |
| 链接与图谱 | `@` 提及、反链、知识图谱可视化 |
| 搜索 | 全局 `⌘K`；中文 jieba + PostgreSQL 全文索引 |
| 标签与评论 | 标签可挂在 KB / 文件夹 / 文档；支持段落级评论 |
| 导出 | MD / HTML / PDF / Word / 整站 zip（Celery 异步） |
| 博客 | 匿名阅读、4 套主题、纸张样式、归档、标签云、RSS |
| AI | 续写 / 润色 / 翻译等 8 种操作；流式输出；后台 `/admin/ai` 配置 |
| 视觉 | 博客宣纸朱砂风；后台玄黑玻璃风；自制图标与 PWA |

## 三分钟上手

1. **启动服务**：`docker compose up -d` → 后端 `runserver 8002` → 前端 `pnpm dev`（3001）→ 可选 Celery worker  
2. **登录后台**：http://localhost:3001/admin ，创建知识库（左侧 `+`）  
3. **写文档**：进入文档编辑器；输入 **`/`** 打开命令面板（插入标题、表格、AI 等）  
4. **发布**：在侧栏或头部将文档设为「已发布」+「公开」，即可在博客首页看到  
5. **AI**：选中文字 → 紫色 **✨** → 选择润色 / 翻译等（需在后端配置 `ANTHROPIC_API_KEY`）

种子本指南（含本文与详细版）：

```bash
cd backend && python manage.py seed_architecture_kb
```
